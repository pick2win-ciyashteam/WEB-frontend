PICK2WIN · OFFICIAL POSITIONING STATEMENT

PICK2WIN is a structured virtual team configuration and workflow assistance platform designed to help users generate organized team combinations quickly after official lineups are released.

The platform provides:

  • ⚙️structured team generation workflows
  • 📊balanced configuration structures
  • 👑Captain & Vice-Captain configuration support
  • 🧩multi-team configuration assistance

Generated outputs are reference structures created from user-selected inputs and configurations.

Users independently decide:

  • player selections
  • substitute selections
  • mandate selections
  • Captain & Vice-Captain selections
  • how generated outputs are used

PICK2WIN does not make decisions on behalf of users.

FOOTBALL API PARTNER

Football data sourced from our third-party football API provider.

All starting XIs, substitute lists, player metadata, fixture timings, and live match status come from a third-party football API provider — covering exactly the leagues PICK2WIN operates across. When the provider publishes a starting XI, our UCT window opens. When the provider updates a substitute, our engine reflects it. PICK2WIN does not maintain its own lineup database.
